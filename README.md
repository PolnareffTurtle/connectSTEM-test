# ConnectSTEM Game

This is a game created by students in CS @ [ConnectSTEM](https://connectstem.org/)!

Check out our game online [here](https://polnareffturtle07.itch.io/ghost-survival)

## How to run locally

### 1. Open Terminal and navigate to a directory where you want your local repository.
```
cd [your/filepath/here]
```

### 2. Clone the repository + navigate to it.
```
git clone github.com/PolnareffTurtle/connectSTEM-test
cd connectSTEM-test
```

### 3. Create + activate a virtual environment (on mac, use python3 instead).
```
python -m venv .venv
source .venv/bin/activate
```

### 4. Install dependencies. 
```
pip install -r requirements.txt
```

### 5. Run the game! (on mac, use python3 instead).
```
python main.py
```
